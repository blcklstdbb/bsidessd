## masOS VM Prep
### Install homebrew in your macOS VM
* From [https://brew.sh/](https://brew.sh/):

```$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"```

### Install git
* Use Homebrew to install git

```$ brew install git```
